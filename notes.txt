Connect GitHub to Netlify
This is where the magic happens. Once connected, Netlify will "watch" your GitHub.

Log in to Netlify: Go to app.netlify.com.

Add New Site: Click Add new site > Import an existing project.

Select GitHub: Connect your GitHub account and find the my-portfolio repository.

Deploy: Leave the "Build settings" empty (since this is plain HTML/JS) and click Deploy my-portfolio.
